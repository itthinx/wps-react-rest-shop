This is the WooCommerce Product Search - React & REST API Example Shop Implementation

  • An example Shop built with React which uses the REST API of the WooCommerce Product Search extension.
  • For details on the REST API endpoints used in this example, visit the REST API section of the documentation for the WooCommerce Product Search extension.
  • You can fork the repository to derive your own example or build upon it.
  • This is not meant to be an example of a fully functional shop or to look pixel-perfect and appealing. Rather, it provides a simple example that illustrates and helps to understand how to work with React components and the endpoints that the extension provides.
